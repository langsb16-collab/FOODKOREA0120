# K-Taste Route - 이미지 사용 가이드

> **목적**: 해외 관광객 설득 + Eatwith 스타일 감성 기준
> **단순 음식 사진 ❌ → 사람과 식탁이 함께 나오는 현지 체험 느낌 ⭕**

---

## 📸 **추천 사진 콘셉트**

### ✅ **이런 사진을 사용하세요**

#### 1. **사람 + 식탁이 함께 나오는 장면**
- 현지인이 음식을 먹고 있는 자연스러운 모습
- 가족/친구들이 함께 식사하는 장면
- 요리사의 손길이 보이는 조리 과정
- 시장에서 장을 보는 사람들

**예시:**
```
✅ "할머니가 김치찌개를 푸고 있는 장면"
✅ "친구들이 돼지고기를 구워먹는 모습"
✅ "시장 아주머니가 빈대떡을 부치는 손"
✅ "가족이 한상 차려진 식탁에 둘러앉은 모습"
```

#### 2. **자연광, 약간 어두워도 OK**
- 스튜디오 조명이 아닌 자연광
- 창가에서 들어오는 햇빛
- 저녁 식사의 따뜻한 조명
- 약간 언더 노출되어도 진정성이 느껴지면 OK

**이유:** 
- 진짜 로컬 맛집의 느낌
- 광고 사진이 아닌 실제 경험
- 해외 관광객이 "나도 저기 가고 싶다" 느낌

#### 3. **현지에서 함께 먹는 느낌**
- "음식만" 찍은 사진 ❌
- "사람 + 음식 + 공간" 함께 ⭕
- 식탁 분위기, 그릇, 손, 표정

## ❌ **피해야 할 사진**

### 1. **스튜디오 촬영 음식**
```
❌ 검정 배경에 음식만 클로즈업
❌ 너무 깔끔하고 인공적인 플레이팅
❌ 백화점 식당 같은 고급스러운 조명
```

### 2. **과도한 색보정**
```
❌ 채도가 너무 높아서 비현실적
❌ HDR 필터로 너무 선명한 사진
❌ 인스타그램 과다 보정
```

### 3. **백화점·프랜차이즈 느낌**
```
❌ 체인점 메뉴판 사진
❌ 너무 정돈된 인테리어
❌ 상업적인 광고 느낌
```

## 🎯 **페이지별 이미지 사용 기준**

### 1. **Hero 섹션 (메인 페이지 상단)**
**목적:** 첫인상, 감성 전달

**추천:**
- 사람들이 한상 가득 차린 식탁에 둘러앉은 모습
- 시장에서 음식을 사는 관광객
- 노포 할머니가 음식을 만드는 장면

**크기:** 
- PC: 1920x1080px (16:9)
- Mobile: 750x1334px (9:16)

**예시 프롬프트 (AI 이미지 생성용):**
```
Natural light, Korean family dining together at local restaurant, 
candid moment, not commercial photography, warm atmosphere, 
traditional Korean table setting with multiple dishes
```

---

### 2. **맛집 카드 (Restaurant Card)**
**목적:** 맛집의 분위기 전달

**추천:**
- 식당 외관 (간판, 입구)
- 주인장이 음식을 내오는 모습
- 손님이 음식을 먹는 장면
- 특징적인 음식 1~2개 (사람 손 포함)

**크기:** 4:3 비율
- PC: 800x600px
- Mobile: 600x450px

**금지:**
- 음식만 클로즈업
- 스튜디오 조명
- 너무 깔끔한 플레이팅

---

### 3. **미식 투어 패키지 (Package Card)**
**목적:** 여행의 경험 전달

**추천:**
- 관광객이 음식을 먹으며 웃는 모습
- 요리 체험 장면
- 시장 투어 장면
- 지역 풍경 + 음식

**크기:** 16:9 비율
- PC: 1200x675px
- Mobile: 800x450px

---

### 4. **후기 섹션 (Reviews)**
**목적:** 신뢰성 전달

**추천:**
- 실제 방문객의 셀카 (음식과 함께)
- 식당에서 찍은 자연스러운 사진
- 단체 사진

**크기:** 1:1 비율 (정사각형)
- 400x400px

---

## 🔍 **이미지 소싱 방법**

### 1. **무료 이미지 사이트**
추천 사이트:
- **Unsplash** (unsplash.com)
- **Pexels** (pexels.com)
- **Pixabay** (pixabay.com)

검색 키워드 예시:
```
"korean food dining"
"local restaurant korea"
"korean market"
"korean family meal"
"traditional korean food"
```

### 2. **AI 이미지 생성**
추천 도구:
- **Midjourney**
- **Stable Diffusion**
- **DALL-E 3**

프롬프트 예시:
```
natural light photography, korean local restaurant, 
people dining together, traditional korean food, 
candid moment, not commercial, authentic atmosphere,
wide angle shot, film photography style
```

### 3. **직접 촬영**
권장 사항:
- 스마트폰 사용 가능 (자연스러움 중요)
- 자연광 활용
- 사람이 음식을 먹는 모습 포함
- 너무 정돈하지 말고 자연스럽게

## 📏 **이미지 최적화 기준**

### 파일 형식
- **JPG**: 사진 (품질 85%)
- **WebP**: 권장 (용량 30% 절감)
- **PNG**: 로고, 아이콘만

### 파일 크기
- **Hero 이미지**: 최대 500KB
- **카드 이미지**: 최대 200KB
- **썸네일**: 최대 50KB

### 최적화 도구
- **TinyPNG** (tinypng.com)
- **ImageOptim** (Mac)
- **Squoosh** (squoosh.app)

## 🎨 **이미지 편집 가이드**

### 1. **색보정**
```css
/* 권장 설정 */
brightness: 100~110%  (약간 밝게)
contrast: 100~105%    (약간 선명하게)
saturation: 90~100%   (자연스럽게)
```

### 2. **크롭**
- 16:9: Hero, 패키지 카드
- 4:3: 맛집 카드
- 1:1: 후기, 프로필

### 3. **필터**
❌ 피해야 할 필터:
- HDR
- Vivid
- High Contrast

✅ 권장 필터:
- Natural
- Film
- Warm

## 🎯 **핵심 원칙 (AI/디자이너에게 전달용)**

```
1. 음식만 찍지 말고, 사람 + 음식 + 공간을 함께 담는다.
2. 자연광을 활용하고, 약간 어두워도 진정성이 느껴지면 OK.
3. 스튜디오 조명, 과한 색보정, 백화점 느낌은 피한다.
4. "현지에서 함께 먹는 느낌"을 전달한다.
5. 상업적인 광고 사진이 아닌, 실제 경험을 담는다.
```
